This website is a collection of original puzzles, for the sake of practicing puzzle design, and to create a type of website that I would personally like to see more of. For each puzzle, there is both a hint and solution that will help you along the way if necessary. The puzzles require out of the box thinking, and pull inspiration from many different types of puzzles, so try to think differently from usual if you get stuck.
